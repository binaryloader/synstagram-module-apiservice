# synstagram-module-apiservice

API service module for the Synstagram iOS app, distributed as a CocoaPods pod.

## Modules

### Auth
Authentication service including login and logout models, network targets, and service layer.

## Dependencies

- BinaryloaderExtensions 1.0.2
- BinaryloaderNetwork 1.0.4

## Installation

Add the private spec repo and pod to your `Podfile`:

```ruby
source 'https://github.com/binaryloader/synstagram-module-cocoapods-specs.git'

pod 'APIService/Auth'
```
